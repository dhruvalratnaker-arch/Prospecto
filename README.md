# Prospecto

Prospecto is an AI-powered scholarship matching platform that helps students find scholarships they actually qualify for and reduce their tuition gap.

## Features

- AI-powered scholarship matching
- Scholarship match scoring system
- Tuition gap analysis
- Deadline reminders
- Automated scholarship data collection

## Tech Stack

- Java
- Spring Boot
- REST APIs
- JSON data storage

## How It Works

Students input information such as GPA, income, major, and extracurricular interests. Prospecto analyzes this data and generates personalized scholarship matches ranked by eligibility score.

- Real-time scholarship database expansion
